（二）删除表

删除程度可从强到弱如下排列：

  1. drop table 表名; 直接将表格删除，无法找回。
  2. truncate table 表名; truncate 是删除表中所有数据，但不能与where一起使用
  3. delete from 表名; 删除表中数据，但可以与where连用，删除特定行
