（一）什么是完整性约束？

为了防止步符合规范的数据存入数据库，在用户对数据进行增删改操作时，MySQL提供一种机制来检查数据库中的数据是否满足规定的条件，确保数据的准确性和一致性。

（五）已存在的表添加约束

  • 一般：
    alter table 表名 add constraint 自定义约束名 约束类型;

  • 特殊：对于想给主键增加auto_increment约束（只能使用列级约束）
    alter table 表名 modify 字段名 类型 auto_increment;
